# Liberty Arch Install script
My Arch Linux installation script, which makes installation as simple as possible.
- u1.sh, u2.sh − Installation on UEFI with the BTRFS file system.
- b1.sh, b2.sh − Installation on BIOS with the BTRFS file system.
- b1e4.sh, b2.sh − Installation on BIOS with the EXT4 file system.

Later I will probably write detailed instructions on how it works.
